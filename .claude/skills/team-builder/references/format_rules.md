# フォーマットルール・メカニクス定義

## メカニクス選択

構築開始時に有効なメカニクスを選択する（複数選択可、未選択 = バニラルール）。

### mega（メガシンカ）
- **効果**: 1体のポケモンがバトル中にメガシンカ可能。種族値・特性・タイプが変化
- **DB判定**: `pokedex` テーブルの `mega_evolution` カラムが非空のポケモンにメガフォームが存在
- **制約**: チーム内1体のみメガシンカ可能
- **メガフォームのデータ取得**: `pokedex_name` で `mega_evolution IS NOT NULL` のエントリを検索し、対応する `local_pokedex_status`/`local_pokedex_type` を取得
- **対象世代**: XY, ORAS, SM, USUM, Legends Z-A（予定）

### gigantamax（キョダイマックス）
- **効果**: 対象ポケモンがダイマックス時に専用キョダイマックス技を使用可能
- **DB判定**: `pokedex` テーブルの `gigantamax` カラムが非空のポケモンにキョダイフォームが存在
- **制約**: ダイマックスはチーム内1体のみ（3ターン限定）
- **対象世代**: SWSH

### zmove（Zワザ）
- **効果**: 全ポケモンがZクリスタルを持つことで1回限りのZ技を使用可能
- **DB判定**: DB該当なし（全ポケモン対象のシステムメカニクス）
- **Z技威力換算表**:

| 元技の威力 | Z技威力 |
|-----------|---------|
| 0-55 | 100 |
| 60-65 | 120 |
| 70-75 | 140 |
| 80-85 | 160 |
| 90-95 | 175 |
| 100 | 180 |
| 110 | 185 |
| 120-125 | 190 |
| 130 | 195 |
| 140+ | 200 |
| 変化技 | 追加効果付き |

- **対象世代**: SM, USUM

### terastal（テラスタル）
- **効果**: 1体のポケモンのタイプを選択したテラスタイプに変更。テラスタイプ一致技にSTABボーナス
- **DB判定**: DB該当なし（全ポケモン対象のシステムメカニクス）
- **制約**: チーム内1体のみテラスタル可能（バトル中1回）
- **STAB計算**:
  - 元タイプ一致 + テラスタイプ一致: 2.0倍
  - テラスタイプのみ一致: 1.5倍
  - 元タイプ一致のみ（テラス後失う）: 1.5倍維持
- **防御面**: テラスタル後は単タイプとして弱点・耐性を再計算
- **対象世代**: SV, Pokemon Champions

## フォーマットプリセット

| プリセット名 | 有効メカニクス | レギュレーション | 備考 |
|-------------|---------------|-----------------|------|
| SV ランクマ | terastal | — | 現行レギュレーション |
| Pokemon Champions | terastal, mega | M-A | `--version champions --regulation M-A` |
| SWSH | gigantamax | — | 過去世代 |
| SM/USUM | zmove, mega | — | 過去世代 |
| バニラ | なし | — | メカニクスなし |

### Pokemon Champions 固有ルール

- **レギュレーション制**: シーズンごとに使用可能ポケモンが変わる。現在は M-A
- **M-A**: 213体 + メガシンカ59種が使用可能
- **CLI**: `--version champions` で Champions 版データを使用。`--regulation M-A` でレギュレーション外ポケモンを制限
- **技・種族値**: Champions 独自の技性能・習得リストを使用（既存作品と異なる場合がある）
- **メガシンカ**: Champions 新規メガを含む。チーム内1体のみ

## 使用制限ポケモン

フォーマットによって使用制限が異なる。

### SV非準伝説ルールで除外される代表例
- 準伝説: レジ系, 三鳥, 三犬, レジ系, ラティ兄妹, UB, etc.
- 伝説: ミュウツー, ルギア, ホウオウ, etc.
- 幻: ミュウ, セレビィ, etc.

### Champions M-A
- M-A リスト外のポケモンは使用不可（`pkdx search --regulation M-A` で確認可能）
- 伝説・幻は M-A リストに含まれない
